# xdcc-telegram-bot

Bot Telegram: /szukaj -> wybór wyniku -> wybór ścieżki -> pobieranie -> potwierdzenie.

## Dev (lokalnie)
1. Utwórz `.env` na bazie `.env.example`
2. Zainstaluj dependencies
3. Uruchom:
   python -m app.main

## Docker
- Dockerfile buduje obraz
- compose.yml uruchamia kontener z env i wolumenami
